ODP Extension HELP!?

Document that describes every feature of this add-on. ( The is no ODPExtension's book, sorry. )

ODP Tools

  • 'From' category - Menu with shortcuts to tools related to the selected category.
    • Auto 'From' category - Same as "'From' category" but will pop-up automatically when a category is highlighted.
  • 'From' editor - Menu with shortcuts related to the selected editor.
  • 'From' URL - Menu with shortcuts related to the selected URL.
  • Local category finder - Fast search tool to find any category of the Open Directory.
  • Semi WorldLinkerate URL - Sends to unreview the selected URL to the selected categories.
  • Places Menu - Shortcuts to several ODP Places.
  • Informative Pop-up - Shows category, title, description and the categories where the current site is listed.
  • ODP Notes buttons - Internal URL note manager and form automation.
  • Category abbreviations - Abbreviate the categories of the ODP Extension menus and in the browser tab title.
  • Category browser - Category history manager with real time category navigation. It's possible append 'searches' of the Local category finder.
  • Category navigator toolbar - Features navigation of ancestors, parents, childs and sister categories.
  • ODP Pages tweaks - CSS Styles and form automation.
  • Listing highlighting - Check the listing of the selected/all links of a document.

Generic Tools

  • URL Tools menu - Tool that extract data for the selected link and open a new tab with a URL that use that data.
  • Translate - Translate the selected text of a textbox, textarea, document or framed document using Google translation.
  • Word Check - Open a new tab with your favourite dictionary/word reference for the selected text.
  • Frames listing - Context menu showing all URLs of the frames in the current page.
  • FrameSet Advice - Notice when the current document has Framesets painting a red dot in the middle of the 'd'
  • HTTP status check - Check the HTTP status of the selected/all links of a document.
