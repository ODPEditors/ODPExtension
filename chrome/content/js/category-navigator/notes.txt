SEMI DOCUMENTATION OF THIS FOLDER

	NOTES 
		
		the "category navigator" is the full toolbar like a breakcumbs to allow easy category switching/editing
		
		term "sisters cateogires"
			sister categories are categories that are equivalent or stay at the same level
			based on this regular expression 
			Regional/Africa/[^/]+/Arts_and_Entertainment/
				the followin two are equivalents
					Regional/Africa/nigeria/Arts_and_Entertainment/
					Regional/Africa/angola/Arts_and_Entertainment/
					
		term "Ancestors cateogires"
			Ancestors categories are categories that are Ancestors of the current category with the same or similar name
			based on this category
			Regional/Africa/nigeria/Arts_and_Entertainment/
				the followin are Ancestors (a not real example but..)
					Regional/Africa/Arts_and_Entertainment/
					Regional/Arts_and_Entertainment/
					Arts_and_Entertainment/
					
		term "childs cateogires"
			childs categories are categories that are child of the current category with the same or similar name
			based on this category
			Regional/Africa/nigeria/Arts_and_Entertainment/
				the followin are Ancestors (a not real example but..)
					Regional/nigeria/region1/Arts_and_Entertainment/
					Regional/nigeria/region2/Arts_and_Entertainment/
					Regional/nigeria/region2/subregion2/Arts_and_Entertainment/
	FILES 

		categoryNavigatorToolbarUpdate.js

			updates the content of the toolbar base on the focused category/location

		categoryNavigatorToolbarbuttonUpdate.js

			when a button is pressed query the categories.txt database in order to retreive the subcategories and the sisters cateogires
			
		categoryNavigatorToolbarbuttonAncestorsCategoriesUpdate.js
		
			the last category of this toolbar breakcumbs, popups NOT the sister categoires but all the Ancestors with the similar name
		
		categoryNavigatorToolbarbuttonChildsCategoriesUpdate.js
		
			the last category of this toolbar breakcumbs, popups NOT the sister categoires but all the childs with the similar name
			
	GLOBALS FOR INSTANCE
		
		this.shared.categories.ancestors.no - an array holding the categories that not contains ancestors
		this.shared.categories.child.no - an array holding the categories that not contains childs
		this.shared.categories.sisters.no     -an array holding the categories that not contains sisters and subcategories
